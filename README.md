# 项目简介 java77

本项目是一个基于Springboot和Vue的图书管理系统，采用前后端分离的设计模式。

## 技术栈

- Java
- Springboot
- MyBatis
- MySQL
- jQuery
- HTML
- Vue3

## 功能模块

### 后台管理

- 首页
- 管理者管理
  - 图书信息管理
    - 图书信息
    - 图书借阅
    - 图书归还
  - 轮播图管理
  - 图书资讯管理
  - 读者管理
  - 图书分类管理

### 前台界面

- 图书信息展示
- 图书信息推荐
- 图书资讯
- 个人中心

## 系统角色

- 系统管理员
- 图书管理者
- 读者

## 运行环境

- 后端：Java 1.8+
- 前端：现代浏览器
- 数据库：MySQL 5.7+

## 部署步骤

1. 准备运行环境
2. 部署后端服务
3. 部署前端服务
4. 初始化数据库
5. 运行系统
